# fonts

Installs Nerd Fonts to `~/.local/share/fonts/`.

## Variables

| Variable | Default | Description |
|---|---|---|
| `fonts_nerd_fonts_version` | `v3.4.0` | Nerd Fonts release version |
| `fonts_nerd_fonts_url` | GitHub release URL | Download URL for SourceCodePro zip |
| `fonts_install_dir` | `~/.local/share/fonts` | Font installation directory |
